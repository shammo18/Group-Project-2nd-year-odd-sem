Restore Permutation

Problem Link : https://codeforces.com/contest/1208/problem/D

Tags: Data Stucture (segment tree)


Solution:

Let us fill the array with numbers from 1 to 𝑁 in increasing order. 1 will lie at the last index 𝑖 such that 𝑠𝑖=0. Find and remove this index 𝑖 from the array and for all indices greater than 𝑖, reduce their 𝑠𝑖 values by 1. Repeat this process for numbers 2,3,...𝑁. In the 𝑖𝑡ℎ turn, reduce the elements by 𝑖.
To find the last index with value zero, we can use segment tree to get range minimum query with lazy propagation.

Time complexity: 𝑂(𝑁⋅𝑙𝑜𝑔(𝑁))
